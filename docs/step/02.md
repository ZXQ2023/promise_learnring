# 2. 状态封装

Promise 有三种状态：pending、fulfilled、rejected。状态一旦变更就不可逆转，且增加异常捕获。

::: tip 本节要点
新增状态常量 PENDING / FULFILLED / REJECTED。resolve 和 reject 中加入状态守卫，确保只能变更一次。
同时用 try/catch 包裹 executor，捕获同步异常。
:::
