# 14. finally 实现

finally 无论 Promise 成功还是失败都会执行，且不改变最终的值。也是 ES6 扩展，不属于 A+ 规范。

::: tip 本节要点
新增 `finally(onFinally)` 方法，基于 then 实现。
在 fulfilled 路径中：执行 onFinally，然后返回原值。
在 rejected 路径中：执行 onFinally，然后重新 throw 原因。
:::

## 新增 finally 方法

<CodeBlock code="finally(onFinally){
  return this.then(
    (value) => {
      onFinally();
      return value;
    },
    (reason) => {
      onFinally();
      throw reason;
    }
  )
}" title="新增方法" />
