# 8. 重构 then 方法

将 then 方法中的逻辑统一：先定义 handler，再判断状态。如果状态已确定就立即执行，否则等状态变更后执行。

::: tip 本节要点
重构 then 方法，将逻辑统一为：先定义 #handlers 回调函数，然后检查状态。
如果状态不是 PENDING 就立即执行 handler。这样同步和异步场景都能统一处理。
:::
