# 4. 将属性改为私有

使用 ES2022 私有字段语法 `#` 将 state 和 value 设为私有，外部无法直接访问。

::: tip 本节要点
将 `state` 改为 `#state`，将 `value` 改为 `#value`。
私有字段只能在类内部访问，确保状态不被外部篡改。
:::
