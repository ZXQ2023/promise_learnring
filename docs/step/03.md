# 3. 异常抛出处理

用 try/catch 包裹 executor 的执行，同步异常会被捕获并自动调用 reject。

::: tip 本节要点
在 constructor 中，用 `try { executor(resolve, reject) } catch(err) { reject(err) }`
包裹 executor 调用。当 executor 内部抛出同步错误时，自动转为 reject 状态。
:::
